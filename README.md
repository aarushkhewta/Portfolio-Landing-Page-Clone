# 🎨 CAH Personal Portfolio Landing Page Clone

A modern and responsive **Personal Portfolio Landing Page** recreated using **HTML5** and **CSS3**. This project focuses on building a clean, visually appealing UI by recreating a professional portfolio website without using JavaScript.
